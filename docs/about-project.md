# About the Project

**LakaMark.web** is a personal project designed to become the central
platform for my creative work.

The goal of this website is to create a place where I can share my
creations, document my projects, and interact with the community that
follows my work. Instead of relying entirely on social media platforms,
I wanted to build a space that I fully control and that can evolve with
my artistic and technical needs over time.

This project is also an opportunity for me to experiment with software
architecture and continuously improve my development skills.

---

## Why build a personal website?

Today many artists share their work through platforms such as Instagram,
TikTok, or DeviantArt. While these platforms make it easy to reach an
audience, they do not provide a central place that I control.

I wanted to create a platform where I can:

- present my work in a structured way
- share my latest creations
- announce upcoming events
- document ongoing projects
- build engagement with the community

In other words, the website is not only a portfolio. It is intended to
become a central hub for my creative activities.

## Choosing the Right Framework

The goal of this project is not to compare frameworks or claim that one
is better than another. Each framework has its own strengths and design
philosophy.

The framework used in this project was chosen simply because it matches
the needs of this platform and the way I prefer to structure my
applications.

---

## Why not WordPress?

A common question could be why not simply use WordPress to build a
portfolio website.

While WordPress is a powerful platform, the goal of this project goes
beyond creating a simple portfolio.

The platform is designed to become a central place for my creative
activities. I want the flexibility to experiment with new features,
adjust the architecture over time, and build tools that match my
specific needs.

Building a custom application provides the flexibility and control
necessary for the long-term evolution of the project.

---

## Project Philosophy

LakaMark.web represents a long-term personal project where technology,
creativity, and community meet.

The platform is intended to provide a stable and independent place where
my work can be shared and documented over time.

Finally, the project is also part of my personal journey as a developer.
It allows me to experiment, learn new architectural approaches, and
continuously improve the design of the application.
